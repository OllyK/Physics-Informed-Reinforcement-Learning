# Requirements

* install **pytorch** with respect to your system spec
* install **matlab** with your licence
  * currently(Sep, 2022), RCWA depends on MATLAB's RETICOLO library
* install our package `pirl`
    ```shell
    pip install .
    ```

# Run
```shell
python main.py
```
